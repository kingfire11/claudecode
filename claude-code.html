Claude Code API
Назад к инструментам

Claude Code

Официальный CLI

1

Установите Claude Code

Если ещё не установлено, выполните команду в терминале:

Terminal 
npm install -g @anthropic-ai/claude-code
2

Задайте переменные окружения

Для macOS/Linux добавьте в ~/.bashrc или ~/.zshrc:

Bash 
export ANTHROPIC_BASE_URL="https://api.claudecodeapi.cloud"
export ANTHROPIC_API_KEY="YOUR_API_KEY"
3

Для Windows PowerShell

PowerShell 
$env:ANTHROPIC_BASE_URL = "https://api.claudecodeapi.cloud"
$env:ANTHROPIC_AUTH_TOKEN = "YOUR_API_KEY"
4

Или через .claude/settings.json

JSON 
{
  "env": {
    "ANTHROPIC_BASE_URL": "https://api.claudecodeapi.cloud",
    "ANTHROPIC_AUTH_TOKEN": "YOUR_API_KEY"
  }
}
5

Запустите claude в папке проекта

Terminal 
claude
