Claude Code API
Назад к инструментам

Continue

VS Code / JetBrains

1

Установите расширение Continue в VS Code

Найдите "Continue" в маркетплейсе VS Code и установите расширение.

2

Откройте config.yaml

Нажмите значок шестерёнки в расширении Continue → "Open Config". Файл находится в ~/.continue/config.yaml

3

Добавьте модели в config.yaml

YAML 
name: Claude Code API
version: 1.0.0
schema: v1
models:
  - name: Claude Sonnet 4.6
    provider: openai
    model: claude-sonnet-4.6
    apiBase: https://api.claudecodeapi.cloud/v1
    apiKey: YOUR_API_KEY
  - name: Claude Haiku 4.5
    provider: openai
    model: claude-haiku-4.5
    apiBase: https://api.claudecodeapi.cloud/v1
    apiKey: YOUR_API_KEY
