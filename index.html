Установка Claude Code

Пошаговое руководство по установке и настройке Claude Code с подключением к кастомному API Kiro Cheap

Начать установку Требования

Предварительные требования

Перед установкой Claude Code убедитесь, что у вас установлены следующие компоненты:

  • Node.js версии 18 или выше
  • npm (поставляется с Node.js)
  • Доступ в интернет
📦

Проверка установленных версий

Выполните следующие команды для проверки:

Terminal 
node --version
npm --version

Установка на macOS

0

Регистрация и подготовка

Перед установкой необходимо зарегистрироваться и получить API ключ:

  1. Зарегистрируйтесь на kiro.cheap/register
  2. Запомните ваш никнейм — он понадобится для пополнения баланса
  3. Сохраните API ключ из личного кабинета
Зарегистрироваться
1

Установка Homebrew (если не установлен)

Homebrew — пакетный менеджер для macOS, который упростит установку Node.js.

Bash 
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
2

Установка Node.js

Bash 
brew install node
3

Установка Claude Code

Bash 
npm install -g @anthropic-ai/claude-code
4

Настройка settings.json

Откройте файл конфигурации Claude Code:

Bash 
code ~/.claude/settings.json

Или откройте через Finder: ~/Library/Application Support/Claude/settings.json

Вставьте следующую конфигурацию (замените ваш_api_key на ваш ключ из личного кабинета):

JSON 
{
  "$schema": "https://json.schemastore.org/claude-code-settings.json",
  "env": {
    "ANTHROPIC_BASE_URL": "https://api.kiro.cheap",
    "ANTHROPIC_API_KEY": "ваш_api_key",
    "ANTHROPIC_AUTH_TOKEN": "ваш_api_key",
    "CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS": "1",
    "CLAUDE_CODE_EFFORT_LEVEL": "high",
    "CLAUDE_AUTOCOMPACT_PCT_OVERRIDE": "80",
    "CLAUDE_CODE_MAX_OUTPUT_TOKENS": "64000",
    "CLAUDE_CODE_FILE_READ_MAX_OUTPUT_TOKENS": "100000",
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": "1",
    "DISABLE_TELEMETRY": "1",
    "DISABLE_ERROR_REPORTING": "1",
    "DISABLE_AUTOUPDATER": "1",
    "DISABLE_BUG_COMMAND": "1",
    "DISABLE_COST_WARNINGS": "1",
    "DISABLE_NON_ESSENTIAL_MODEL_CALLS": "1",
    "CLAUDE_CODE_DISABLE_FEEDBACK_SURVEY": "1",
    "BASH_DEFAULT_TIMEOUT_MS": "120000",
    "BASH_MAX_TIMEOUT_MS": "600000",
    "BASH_MAX_OUTPUT_LENGTH": "100000",
    "MAX_THINKING_TOKENS": "31999"
  },
  "model": "claude-opus-4.6",
  "skipDangerousModePermissionPrompt": true
}

Важно: Замените ваш_api_key на фактический ключ из личного кабинета Kiro Cheap.

Расположение settings.json

Файл конфигурации settings.json находится в следующих директориях:

ОС Путь к settings.json
macOS ~/.claude/settings.json
~/Library/Application Support/Claude/settings.json
Windows %APPDATA%\Claude\settings.json
Linux ~/.claude/settings.json
~/.config/Claude/settings.json

💡 Совет: После редактирования settings.json перезапустите терминал для применения изменений.

Проверка работоспособности

После настройки убедитесь, что Claude Code работает корректно:

Terminal 
claude --version

Запустите Claude Code и задайте простой вопрос:

Terminal 
claude
> Привет! Расскажи, что ты умеешь делать?

✓ Успех! Если вы получили ответ от Claude — настройка завершена успешно.

Возможные ошибки и решения

Ошибка Причина Решение
command not found: claude Claude Code не установлен или не в PATH Переустановите: npm install -g @anthropic-ai/claude-code
EACCES permission denied Недостаточно прав для установки Используйте sudo или настройте права на npm глобальные пакеты
connection refused Неверный Base URL или проблемы с сетью Проверьте URL: https://kiro.cheap/api
401 Unauthorized Неверный API ключ Получите новый ключ в личном кабинете и обновите конфиг
npm ERR! code EEXIST Конфликт версий или файлов Удалите старую версию: npm uninstall -g @anthropic-ai/claude-code
node: command not found Node.js не установлен Установите Node.js 18+ согласно инструкции для вашей ОС
certificate verify failed Проблемы с SSL сертификатом Обновите сертификаты ОС или временно отключите проверку SSL
Invalid JSON in settings.json Синтаксическая ошибка в JSON Проверьте файл на валидность JSON (запятые, кавычки, скобки)
ANTHROPIC_API_KEY is required Не указан API ключ Добавьте ключ в settings.json: "ANTHROPIC_API_KEY": "ваш_ключ"

Часто задаваемые вопросы

Kiro Cheap — это кастомный API-прокси для Claude Code, который предоставляет доступ к моделям Anthropic по более доступным ценам. Сервис выступает промежуточным звеном между клиентом и официальным API Anthropic.

Зарегистрируйтесь на kiro.cheap/register, пополните баланс и создайте API ключ в личном кабинете.

Да, Kiro Cheap использует официальное API Anthropic. Ваши данные не сохраняются и не передаются третьим лицам. API ключ используется только для аутентификации и тарификации запросов.

Да, с Kiro Cheap вы платите только за фактически использованные токены. Никакой ежемесячной подписки не требуется.

Доступны все модели Anthropic, включая новейшие:
Claude Sonnet 4.6, Claude Opus 4.6, Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Haiku. Выбор модели указывается в settings.json (параметр model) или при запуске через флаг --model.

Для обновления выполните команду: npm update -g @anthropic-ai/claude-code. Рекомендуется регулярно обновляться до последней версии.

Проверьте: 1) Подключение к интернету, 2) Корректность Base URL, 3) Наличие средств на балансе, 4) Статус сервиса на странице Kiro Cheap. Также попробуйте перезапустить терминал.

Да, Claude Code полностью работает в терминале и не требует графического интерфейса. Идеально подходит для удалённых серверов и CI/CD пайплайнов.

Откройте файл settings.json и измените значения ANTHROPIC_API_KEY и ANTHROPIC_BASE_URL. Путь к файлу:
macOS/Linux: ~/.claude/settings.json
Windows: %APPDATA%\Claude\settings.json

Вся статистика использования, история запросов и детализация расходов доступны в личном кабинете на kiro.cheap после авторизации.
