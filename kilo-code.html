Kilo Code

VS Code расширение

⚠️ Изображения не поддерживаются

Kilo Code, OpenCode, Cline / Roo и прочие подобные форки не передают изображения кастомным провайдерам через OpenAI-совместимый и Anthropic эндпоинты — картинки просто не уходят в запрос. Это ограничение самих инструментов, а не нашего API. В Cursor и Claude Code зрение работает корректно.

Что такое Kilo Code?

Kilo Code — это AI-агент для VS Code и JetBrains, который умеет писать код, запускать команды, редактировать файлы и работать со всем проектом. Это форк Roo Code с улучшенным UX и дополнительными функциями. Бесплатное расширение, работает с любым API-провайдером.

1

Установите расширение Kilo Code в VS Code

Найдите "Kilo Code" в маркетплейсе VS Code или JetBrains и установите расширение.

2

Откройте настройки расширения

Нажмите на иконку Kilo Code на боковой панели. В поле "API Provider" выберите "Anthropic".

3

Введите данные подключения

https://api.claudecodeapi.cloud
YOUR_API_KEY
4

Выберите модель и начните работу

В выпадающем списке выберите нужную модель — claude-opus-4.6, claude-sonnet-4.6, claude-haiku-4.5 и другие.
