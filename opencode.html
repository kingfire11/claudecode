Claude Code API
Назад к инструментам

OpenCode

Терминальный агент

⚠️ Изображения не поддерживаются

Kilo Code, OpenCode, Cline / Roo и прочие подобные форки не передают изображения кастомным провайдерам через OpenAI-совместимый и Anthropic эндпоинты — картинки просто не уходят в запрос. Это ограничение самих инструментов, а не нашего API. В Cursor и Claude Code зрение работает корректно.

1

Установите OpenCode

Terminal 
npm install -g opencode-ai
2

Создайте или отредактируйте конфиг

Файл: ~/.config/opencode/opencode.jsonc

JSONC 
{
  "$schema": "https://opencode.ai/config.json",
  "provider": {
    "anthropic": {
      "options": {
        "baseURL": "https://api.claudecodeapi.cloud/v1",
        "apiKey": "YOUR_API_KEY"
      }
    }
  }
}
3

Запустите opencode в папке проекта

Terminal 
opencode
