Назад к инструментам

Windsurf

AI-редактор кода

1

Откройте настройки провайдера

В Windsurf откройте Settings → Windsurf Settings → Providers.

Найдите секцию Anthropic или добавьте нового провайдера.

2

Укажите Base URL и API Key

В настройках провайдера Anthropic укажите:

Настройки 
Base URL: https://api.claudecodeapi.cloud
API Key:  YOUR_API_KEY

Или через файл ~/.codeium/windsurf/config.json:

JSON 
{
  "anthropic": {
    "api_key": "YOUR_API_KEY",
    "base_url": "https://api.claudecodeapi.cloud"
  }
}
3

Выберите модель

В чате Windsurf (Cascade) выберите одну из доступных моделей:

Доступные модели 
claude-opus-4-7
claude-opus-4-6
claude-sonnet-4-6
claude-sonnet-4-5
claude-haiku-4-5
4

Готово!

Откройте Cascade (чат) в Windsurf и начните работать с Claude через наш API. Модель будет использоваться для автодополнения, чата и агентного режима.

Доступные модели

В настройках можно указать: claude-opus-4-7, claude-opus-4-6, claude-sonnet-4-6, claude-sonnet-4-5, claude-haiku-4-5
